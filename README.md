# 📅 Dynamic Timetable Generator

An intelligent web-based timetable generation system that automates the scheduling process for educational institutions. The system generates conflict-free timetables by considering faculty availability, subject constraints, and scheduling rules, significantly reducing the time and effort required for manual timetable creation.

---

## 🚀 Features

- ✅ Automatic timetable generation
- ✅ Faculty availability management
- ✅ Conflict-free scheduling
- ✅ Subject hour allocation
- ✅ Department-wise timetable generation
- ✅ User-friendly interface
- ✅ Secure cloud database using Supabase
- ✅ Responsive design

## 📌 Problem Statement

Creating academic timetables manually is time-consuming and prone to scheduling conflicts such as:

- Faculty clashes
- Overlapping classes
- Incorrect subject hour allocation
- Room scheduling issues

This project automates the process while satisfying predefined scheduling constraints.

---

## ⚙️ How It Works

1. Faculty and subject details are stored.
2. Course constraints are defined.
3. The CSP-based scheduling algorithm generates a valid timetable.
4. Conflicts are checked and eliminated.
5. The final timetable is displayed to the user.
